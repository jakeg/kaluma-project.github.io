var led = board.led(0);

  digitalToggle(led);

Features

Standard compliant
Compliant with ECMAScript 5.1 and 2015(subset) standard, so you can use many tools including Linters (JSHint, ESLint, ...), Transpilers (Babel, TypeScript, ...), Bundlers (Webpack, Rollup, ...).
Powered by JerryScript
Small and fast
Kameleon is very small and efficient which can be run on microcontrollers having about 300KB ROM and 64KB RAM. User program can be stored in the in-chip flash memory without SD card or additional memory.
Event-driven programming

Supports asynchronous even driven programming model. Just register callback functions on various I/O events without polling in the infinite loop.

REPL mode support

You can evaluate code instantly in interactive REPL mode. Any terminal program supporting ANSI/VT100 can connect to Kameleon compatible boards.

Open source
Kameleon is an open source project includes JavaScript runtime, hardware boards and some tools licensed under Apache License.
Github
Portable and extensible
It is easier to port the runtime on a new kind of microcontroller and add a new native module which can be imported in JavaScript code by require() function.

Getting Started

  • macOS
  • Windows
  • Linux
1
Get a board
Get a Kameleon compatible board.
Buy on Tindie
2
Install CLI
Open a terminal and install Kameleon CLI (Command Line Interface).
Ensure that Node.js/NPM is installed.
You may need to use npm option --unsafe-perm=true
$ sudo npm install -g kameleon-cli
3
Write your code
Write your code with a code editor (Sublime Text, Visual Studio Code, Atom, etc.). 
// index.js
var led = board.led(0);
setInterval(function () {
  digitalToggle(led);
}, 1000);
4
Connect the board
Connect the board to your computer using a USB cable.
Check the serial port where the board is connected
$ kameleon -l
...
/dev/tty.usbmodem?
5
Upload your code
Upload your code to the board using CLI 
$ kameleon write index.js -p /dev/tty.usbmodem?

Main features

  • 38mm x 21mm
  • STM32F411CEU6 Microcontroller
  • 100MHz ARM 32-bit Cortex M4
  • 512KB flash and 126KB RAM
  • 20x GPIO, 5x PWM, 6x ADC, 2x I2C, 2x SPI, 2x UART
  • 2 LEDs (1 for power, 1 for user)
  • 2 Switches (1 for reset, 1 for user)
  • On-board 3.3v 250mA voltage regulator, accepts voltages from 3.5v to 5v
  • Kameleon firmware is pre-programmed
  • 128KB is reserved for user program in flash
