Kameleon

A tiny JavaScript platform for Microcontrollers
v0.9.0-beta 
// Blink LED
var led = board.led(0);
setInterval(function () {
  digitalToggle(led);
}, 1000);

Key features

ECMAScript compliant

Compliant with ECMAScript 5.1 and 2015(subset) standard, so you can use many tools including Linters (JSHint, ESLint, ...), Transpilers (Babel, TypeScript, CoffeeScript, ...), Bundlers (Webpack, Parcel, Rollup, ...) to program Microcontrollers. (Powered by JerryScript)

Small and efficient

Kameleon is very small and efficient which can be run on microcontrollers having about 300KB ROM and 64KB RAM. User program can be stored in the flash memory of microcontroller.

Event-driven programming

Supports asynchronous even driven programming model. Just register callback functions on various I/O events without polling in the infinite loop.

REPL mode support

You can evaluate code instantly in interactive REPL mode. Any terminal program supporting ANSI/VT100 can connect to Kameleon compatible boards.

Getting started

  • macOS
  • Windows
  • Linux
1
Get a board
Get a kameleon-compatible board.
Buy on Tindie
2
Install CLI
Install Kameleon CLI (Command Line Interface).
Ensure that Node.js/NPM is installed.
$ npm install -g kameleon-cli
3
Write your code
Write your code with Code Editor (Sublime Text, Visual Studio Code, Atom, Brackets, etc.). 
// index.js
var led = board.led(0);
setInterval(function () {
  digitalToggle(led);
}, 1000);
4
Connect the board
Connect the board to your computer using a USB cable.
Check the serial port where the board is connected
$ ls /dev/tty.*
/dev/tty.Bluetooth-Incoming-Port
/dev/tty.usbmodem<n>
5
Upload your code
Upload your code to the board using CLI 
$ kameleon write index.js -p /dev/tty.usbmodem<n>
Use in REPL mode
